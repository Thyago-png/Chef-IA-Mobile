# 🍳 Chef-IA

Javascript com React Native para o desenvolvimento de uma aplicação mobile. Um chefe integrando inteligência artificial que gera receitas com base em ingredientes.

## 🛠️ Tecnologias Utilizadas
* **React Native** com **Expo**
* **Axios** (Para requisições HTTP)
* **Groq API / Llama 3.1** (Modelo de IA para criação das receitas)

## 🚀 Como Executar o Projeto Localmente

### Pré-requisitos
* Node.js instalado na máquina.
* Aplicativo **Expo Go** instalado no celular.

### Passo a Passo

1. **Instalar dependências:**
   Abra o terminal na pasta do projeto e rode o comando:
```bash
   npm install

**Configure sua API Key:**
   Abra o arquivo `App.js` e substitua `MySecretKey` pela sua chave de API.
